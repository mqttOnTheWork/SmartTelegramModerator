# История версий

Краткий журнал релизов проекта. Полный changelog на английском — в
[`CHANGELOG.md`](CHANGELOG.md).

## v1.0.0
Первый стабильный релиз. Мониторинг: метрики Prometheus и эндпоинт `/metrics`,
сервисы Prometheus и Grafana в docker-compose. Финальная документация.

## v0.9.0
Dockerfile, docker-compose (API + PostgreSQL + Redis) и CI на GitHub Actions
(линт, тесты, сборка образа).

## v0.8.0
Web Dashboard: страница со статистикой модерации и JSON-эндпоинт.

## v0.7.0
REST API на FastAPI с JWT-авторизацией и проверкой ролей.

## v0.6.0
Действия модерации: предупреждения с эскалацией, муты, баны, репутация, роли.

## v0.5.0
ML pipeline: обучение и инференс токсичности с fallback-эвристикой.

## v0.4.0
Движок модерации: фильтры ссылок, мата, рекламы, повтор-спама и флуда.

## v0.3.0
Каркас Telegram-бота и обработчики команд.

## v0.2.0
Слой данных: модели PostgreSQL (SQLAlchemy, async) и клиент Redis.

## v0.1.0
Каркас проекта: конфигурация, логирование, исключения и тесты.
