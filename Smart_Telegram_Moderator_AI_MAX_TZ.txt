================================================================================
1. ЦЕЛЬ ПРОЕКТА
================================================================================


Создать интеллектуального Telegram-бота для автоматической модерации групп.
Проект должен объединять:
- Machine Learning
- NLP
- Telegram Bot API
- FastAPI
- PostgreSQL
- Redis
- Docker
- CI/CD
- Web Dashboard
- систему логирования
- систему ролей
- масштабируемую архитектуру.
